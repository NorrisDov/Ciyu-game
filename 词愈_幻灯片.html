PART 2 · 关于词愈
故事的起源：一场对话
赵川： “你知道吗，AI的本质就是学习词语之间的关联。我们把海量文本喂给模型，让它学习词间的联系，最终我们判断它的生成是否正确，并加以改进，逐渐地模型就能够说出一句连贯的话、写出完整的文章。这个过程，和人类记忆的形成，底层逻辑可能是一样的。”
赵川： “你父亲的记忆定格在他的中学时期——那是他学习能力最旺盛的时候。他的大脑——似乎在尽力保护着相关的回忆。所以幸运的是，他对新知识仍然有热情。如果我们不让他‘回忆’，而是让他‘学习’——把那些记忆包装成‘训练数据’，让他以‘当AI’的方式重新吸收它们……也许能绕过他的抗拒心理。”
林可： “成功的概率有多大？”
赵川： “我不知道。但人工智能的成功，至少证明了一件事：词与词之间的联系是可以被习得的。我们认为阿尔茨海默症患者也有可能通过同样的方式，重新建立起词语之间的联系。最终是否接受这个实验，需要你的意见。”
林可： “我想让他再叫我一声女儿。”
于是他们制定了一个计划。 既然父亲不愿意接受试图回忆带来的挫败感，那就让他以新的身份来学习这些回忆——一个叫 LINK-1 的人工智能。
这就是这个“游戏”的起源。

PART 3 · 核心设计
一、人工智能 —— 用概率预测模型说话
📖 概念说明
LLM本质上是"下一个词预测器"（next-token prediction）。
给定上文，计算每个候选词的出现概率，选择最高概率的输出。

现实案例：
GPT-4训练数据约13万亿token，通过自注意力机制学习词间关联。你输入"今天天气真"→模型计算"好"(72%), "热"(15%), "差"(8%)→输出"好"。
🎮 游戏对应处理
玩家扮演LINK-1，从候选词中选择最合理的下一个词。

WeightMatrix类实现：
learn(a, b, w) // 记录词对权重 strengthen(a,b,inc) // 强化关联 getWeight(a,b) // 查询关联强度
系统日志实时显示：
[网络统计] 总关联: 25 | 强关联: 8 | 弱关联: 5

PART 3 · 核心设计
大语言模型的训练 —— 用反馈提高输出质量
📖 现实中如何做
大语言模型的训练并非一次成型，而是通过海量反馈循环不断优化：

① 预训练：模型在海量文本上学习词间的统计规律。

② 指令微调（SFT）：用人工标注的高质量问答对，让模型学会遵循指令。

③ 强化学习（RLHF）：人类对模型输出排序打分——
奖励有用/无害的回答，惩罚有害/幻觉内容。

每一次错误输出被纠正，都相当于一次"权重更新"——
正确路径的权重升高，错误路径的权重降低。
🎮 游戏中的对应
游戏中，这个流程被简化为：

1. 玩家选择 → 从候选词中选一个
2. 系统反馈 → 正确则强化权重，错误则显示提示
3. 重新选择 → 玩家换一个选项再次尝试
4. 权重记录 → 系统日志实时更新关联强度

每一次"试错—纠正—确认"，
都是LINK-1的"梯度下降"。

PART 3 · 核心设计
三、阿尔兹海默症 —— 当神经元变成大脑的孤岛
📖 概念说明
阿尔兹海默症的病理核心，是两种异常蛋白质在大脑中的沉积与破坏：

① β-淀粉样蛋白（Aβ）—— 像"错误的黏合剂"，在神经元之间聚集成老年斑，
物理阻挡信号传递，并激活免疫反应攻击周围神经元。

② Tau蛋白—— 本是神经元内部的"骨架支撑"，异常磷酸化后缠结成神经纤维缠结，
使神经元内部的营养运输通道瘫痪，细胞逐渐饿死。

两者联手，从海马体（记忆中枢）开始蔓延至整个皮层。
突触大规模丢失，大脑萎缩，皮质变薄——
最终，神经元的网络变成无法通信的孤岛。
🎮 游戏对应处理
decay() 方法模拟突触退化：
decay() { for(每个关联){ if(30%概率) 权重 -= 10; // 下限为10 } }
第二幕训练"2-4"后触发衰减——
"▓▓▓ 关联丢失 ▓▓▓"
"错误代码：LINK-1 语义退化"

训练内容映射：
记忆→模糊→雾→看不清→恐惧→失去
父亲→背影→远去→告别→眼泪→成长

PART 3 · 核心设计
四、亲情 —— 超越记忆的能力
📖 概念说明
情感记忆比事实记忆存续更久。
杏仁核在疾病早期相对完好——
患者记不住名字，但感受得到"这个人让我安全"。

现实案例：
医学文献记载：一位记不住女儿名字的母亲，会在女儿哭泣时伸手抚摸她的脸；一位无法对话的父亲，听到孙子笑声时会露出微笑。

→ 某些权重被赋予了极高初始值，
在整体衰减时仍保持显著性。
→ 这种"高权重连接"构成了对抗遗忘的最后防线。
🎮 游戏对应处理
权重100的最终守护：
林→可 (weight:100) 可→是 (weight:100) 是→女 (weight:100) 女→儿 (weight:100) 儿→。 (weight:100)
错误选项权重为0——
确保唯一步向"林可是女儿。"

"选择即意义"：
终局乱码选项——玩家无需判断"正确"，
选任一都会推进。
选择的行为本身，
而非选择的内容，才是修复记忆的关键。

结语
"爱，是唯一不褪色的记忆。"
《词愈》不仅仅是一个游戏。
它是一个关于记忆、爱与遗忘的互动寓言。

有些记忆并不是"不记得了"，
而是大脑缺失了调用这份记忆的联系。

高权重的关联——被反复强化、被赋予情感价值的连接——
构成了我们之所以为"我们"的核心。

当连接因疾病而断裂时，
爱——作为最深层的生物编码——
是唯一能够重新激活它们的信号。
